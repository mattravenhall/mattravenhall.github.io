Overview

This is a mini-repository of some tools and utilities I've created for a variety of in and out of lab purposes. An extended list of side projects can be found on the Projects or Posts pages.

Highlights

  • outbreaker: Command line tool for WHO disease outbreak news
  • SV-Pop: Multi-population based SV exploration
  • PfINV: Detecting inversions in long read assemblies
  • bio: Python Bioinformatics Module
  • BasicNeuralNetwork: Mini-tool for building simple neural networks
  • BasicSIRModel: Utility for exploring the SIR model
  • TweetSentiment: Real time hashtag sentiment analysis
  • Gists: Short scripts of miscellaneous purposes.
